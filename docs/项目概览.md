# UAVRoadDetection 项目概览

## 1. 项目简介
UAVRoadDetection 是一套面向公路/道路巡检场景的“无人机采集 + AI 识别 + 病害历史匹配 + 地图可视化”系统原型。
系统支持：
- 无人机设备管理（注册、查询、维护）
- 巡检任务管理（创建、航线点位、启动/结束）
- 无人机图像与遥测数据上报
- 路面病害目标检测（如裂缝/坑槽等，具体类别以模型为准）
- 相邻帧去重与多时相病害匹配（GNN + LoFTR 兜底）
- 地图点位展示、详情查看、实时告警（WebSocket）

## 2. 总体架构
- 前端（Vue3 + Vite）：地图展示、设备管理、任务管理、采集控制台
- 后端（Spring Boot 3 + MyBatis-Plus + Redis + MySQL）：业务接口、文件保存、结果入库、缓存、WebSocket 推送
- AI 服务（FastAPI + Ultralytics YOLO + LoFTR + 自研 GNN 模型）：检测、图像匹配、病害点匹配推理

默认端口：
- 前端：5173
- 后端：8080
- AI 服务：8000
- Redis：6379
- MySQL：3306

## 3. 目录结构（关键）
- backend/            Spring Boot 后端
- frontend/           Vue3 前端
- ai-service/         FastAPI AI 推理服务（YOLO/LoFTR/GNN）
- disease_matching/   病害点匹配模型（训练/评估/推理）
- db/                 MySQL 数据库 dump（可用于导入初始化）
- api-docs.yaml       OpenAPI 接口描述（REST API）

## 4. 运行形态说明（原型特征）
当前版本包含若干“本地绝对路径”配置（尤其是图片保存目录、模型权重路径），部署前需要按文档调整为你本机实际路径，或进一步改造为相对路径/配置化路径（详见 02、08 文档）。
